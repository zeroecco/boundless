## Prerequisites

- [Bun](https://bun.sh)

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/boundless-xyz/boundless.git
```

2. Install dependencies

```bash
bun install
```

3. Start the development server

```bash
bun run dev
```

4. Open your browser and navigate to `http://localhost:5173`
